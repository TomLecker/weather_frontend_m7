# 🌦️ Weather App Vue + Firebase

Aplicación web de clima desarrollada con Vue 3 que permite a los usuarios registrarse, iniciar sesión y guardar ciudades favoritas, mostrando información meteorológica en tiempo real.

## 🚀 Demo

👉 https://TomLecker.github.io/weather_frontend_m7/

## 🧠 Funcionalidades

- Autenticación de usuarios (Firebase)
- Gestión de favoritos
- Preferencias de temperatura
- Rutas protegidas
- Consumo de API de clima

## 🛠️ Tecnologías utilizadas

- Vue 3
- Pinia
- Vue Router
- Firebase Authentication
- Cloud Firestore
- OpenWeather API
- Vuetify

## ⚙️ Instalación

1. Clonar el repositorio

git clone https://github.com/TU-USUARIO/weather_frontend_m7.git

2. Instalar dependencias

npm install

3. Configurar variables de entorno

Crear un archivo `.env` basado en `.env.example`

4. Ejecutar el proyecto

npm run dev


## 🔐 Variables de entorno

Este proyecto utiliza variables de entorno para proteger las claves de API.

Ejemplo:

VITE_FIREBASE_API_KEY=
VITE_WEATHER_API_KEY=
